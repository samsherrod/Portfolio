Average Face Filter - Image Processing Tool

Released March 2022
Student Solo Graphics Programming Project
Role: Solo Developer

This is an image processing tool that allows you to blend multiple photos of faces together. Press the "D" key to load a random image of someone's face. While the "lerp" (blend) checkbox is on, adjust the slider to see the effect. While the slider bar is at 100%, every image will look the same.

This is a tool I made for a graphics programming course at the University of London using p5.js from scratch.

Technologies used:

  • JavaScript
  • p5.js
  • Images provided by UoL
