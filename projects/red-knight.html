Red Knight - 2D Action Platformer

In Development - Expected Release March 2023
Student Group Games Development Project
Role: Lead Developer

Red Knight is a 2D action platformer centered around a vampire hunter seeking revenge against Dracula. It is currently in development being built by myself and 3 other students with the Unity game engine. It is inspired by games such as Megaman and Enter the Gungeon. A unique aspect of the it is that it is both a 2D action platformer and a top down action game similar to Zelda II: The Adventure of Link.

This is a group project made for University of London's Game Development class. The game will be finished and playable on WebGL in March 2023.

I have more experience with Unity than the rest of my team, and my role in the development phase has mainly been to support the team. This has included:

  • building persisting scripts including a game manager, an input manager, and a UI manager
  • creating a tilemap system with rule tiles for teammates to quickly prototype new levels in a flexible way
  • designing game mechanics
  • setting up the controls using Unity's new Input System
  • setting up animation state machine
  • creating our GitHub repository
  • managing all of our files including our GDD, references, weekly tasks lists, features and more on our Google Drive.

Technologies used:

  • C#
  • Unity 2019.4.40f1
  • Various art and sound assets from Unity Asset Store and Itch.io
  • Google Drive for private collaboration
