Budget Planning Mobile App

Released September 2023
Student Mobile Development Project

This was a mobile app I made for the University of London. It allows a user to create withdrawl and deposit transactions to help users track their budgets. Users can add, update, or delete each transaction. The data is stored on a realtime database using Google Firebase. Handles errors if newly added or updated transactions are not created correctly, or if the database is not working correctly. Users can also view any transactions in the last 7 days by clicking on the Recents button.

Technologies used:

  • JavaScript
  • React Native
  • Expo
  • Google Firebase Backend-as-a-service (Baas)
